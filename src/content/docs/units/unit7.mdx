## Introducción 📜

En esta unidad vas a realizar una introducción práctica a la programación gráfica moderna. 
Usarás una plantilla de proyecto preconfigurada para Visual Studio con C++, GLFW, GLAD y GLM 
para explorar cómo se crean gráficos acelerados por hardware con OpenGL. Nos centraremos en 
entender el **pipeline programable** analizando y modificando ejemplos funcionales, culminando 
con algunos problemas simples y una modificación de un caso de estudio. Dado el tiempo limitado, 
nos enfocaremos en los conceptos esenciales para que puedas empezar.

## ¿Qué aprenderás en esta unidad? 💡

*   **Navegar un proyecto OpenGL básico:** entenderás la estructura y el ciclo de vida de una aplicación gráfica simple.
*   **Identificar el rol de los Shaders:** modificarás vertex y fragment shaders básicos (GLSL) y entenderás su impacto.
*   **Comprender el flujo de datos:** analizarás cómo los datos de vértices (VBOs) se conectan a los shaders 
a través de la configuración de estado (VAOs).
*   **Usar Uniforms:** pasarás datos dinámicos (como el tiempo o colores) desde C++ a tus shaders.
*   **Aplicar conocimientos modificando código existente:** reforzarás tu aprendizaje alterando ejemplos 
funcionales y el caso de estudio principal.
